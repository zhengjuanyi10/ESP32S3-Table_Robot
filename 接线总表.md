# 桌面机器人模块接线总表

> 最终架构：ESP32-S3 → TCA9548A（I2C 扩展）→ 传感器 / OLED / I2C 视觉识别模块 / 触摸 / UART 扩展 / 小车控制。
>
> 需要额外准备：SC16IS752 双 UART 模块、PCA9685 PWM 模块、TB6612FNG 双电机驱动模块、ADS1115 四路 ADC 模块（用于两个 MAX9814）。
>
> 所有电源的 GND 必须共地。ESP32 GPIO 只能接 3.3V 逻辑。

## 1. ESP32-S3 原生 GPIO 总览

| 功能 | GPIO | 连接对象 |
|---|---:|---|
| I2C SDA | GPIO8 | TCA9548A SDA |
| I2C SCL | GPIO9 | TCA9548A SCL |
| 舵机总线 TX | GPIO17 | ZLink RX |
| 舵机总线 RX | GPIO16 | ZLink TX |
| I2S 麦克风数据输入 | GPIO1 | INMP441 SD |
| I2S 位时钟 | GPIO19 | INMP441 SCK、MAX98357 BCLK |
| I2S 左右声道时钟 | GPIO20 | INMP441 WS、MAX98357 LRC |
| I2S 音频数据输出 | GPIO41 | MAX98357 DIN |
| TFT SPI 时钟 | GPIO4 | 3.5 寸屏 SCK / CLK |
| TFT SPI MOSI | GPIO5 | 3.5 寸屏 MOSI / SDI |
| TFT SPI CS | GPIO6 | 3.5 寸屏 CS |
| TFT SPI DC | GPIO7 | 3.5 寸屏 DC / RS |
| TFT SPI 复位 | GPIO10 | 3.5 寸屏 RST / RESET |
| TFT 背光控制 | GPIO11 | 外部三极管/MOSFET驱动控制端，不直接带 BL/LED |
| 触摸复位 | GPIO12 | 3.5 寸屏触摸 RST |
| 触摸中断 | GPIO13 | 3.5 寸屏触摸 NT / INT |
| 串口日志 | GPIO43 / GPIO44 | 板载 USB-UART（COM3），不接外设 |

不可用于普通 3.3V 外设：GPIO26 至 GPIO37、GPIO47、GPIO48。
启动配置脚尽量不接外设：GPIO0、GPIO3、GPIO45、GPIO46。
板载 RGB LED：GPIO38。当前预留可用：GPIO2、GPIO14、
GPIO15、GPIO18、GPIO21、GPIO39 至 GPIO42。

本机为 `ESP32-S3-WROOM-2-N32R16V`：GPIO47、GPIO48 属于 1.8V
VDD_SPI 电源域，不能直连 3.3V 外设逻辑；DevKitC-1 v1.1 的 GPIO38
连接板载 RGB LED。GPIO0、GPIO3、GPIO45、GPIO46 是启动配置脚。
GPIO19、GPIO20 分配给 I2S 后不再用于原生 USB。本项目使用 GPIO43/44
的 USB-UART 下载和查看日志，因此这一复用可以接受。GPIO39 至 GPIO42
不再接 TFT，保留给外部 JTAG或后续普通 GPIO。

## 2. I2C 设备总线架构

多数 I2C 设备地址不冲突，直接挂 GPIO8/GPIO9 共享总线。

### 直连 GPIO8/GPIO9（不经 TCA）

| 设备 | I2C 地址 | 说明 |
|---|---|---|
| DS3231 RTC | 0x68 | |
| SHT30 温湿度 | 0x44 | |
| 3.5 寸屏电容触摸 | 0x38 / 0x5D / 0x14 / 0x15 | 自动检测 |
| MAX30102 心率血氧 | 0x57 | |
| OLED | 0x3C | |
| I2C 视觉识别模块 | 按模块标注 | 地址通常不与上述冲突 |

以上所有设备的 SDA 并到 GPIO8，SCL 并到 GPIO9，VCC 接 3V3，GND 共地。

### TCA9548A I2C 扩展板（隔离地址冲突设备）

仅以下三组存在地址冲突的设备通过 TCA9548A 隔离：

| TCA 通道 | 模块 | I2C 地址 |
|---|---|---|
| CH0（SD0 / SC0） | SC16IS752 双 UART 扩展 | 0x48 |
| CH2（SD2 / SC2） | INA219 电源监测 | 0x40 |
| CH3（SD3 / SC3） | ADS1115 四路 ADC | 0x48 |
| CH7（SD7 / SC7） | PCA9685 小车 PWM | 0x40 |

> INA219(0x40) 与 PCA9685(0x40) 冲突 → 分到 CH2 / CH7。
> ADS1115(0x48) 与 SC16IS752(0x48) 冲突 → 分到 CH3 / CH0。

### TCA9548A 与 ESP32-S3 接线

| TCA9548A 引脚 | 连接到 |
|---|---|
| VCC / VIN | ESP32 3V3 |
| GND | ESP32 GND |
| SDA | ESP32 GPIO8 |
| SCL | ESP32 GPIO9 |
| A0、A1、A2 | GND（地址 0x70） |
| RST / RESET（若有） | 3V3 |

| INMP441 引脚 | 连接到 |
|---|---|

| GND | GND |
| SCK | ESP32 GPIO19 |

| SD | ESP32 GPIO1 |

### MAX98357 功放和喇叭

| MAX98357 引脚 | 连接到 |
|---|---|
| VIN / VCC | 外部稳定 5V |
| GND | GND |
| BCLK | ESP32 GPIO19 |
| LRC / LRCK / WS | ESP32 GPIO20 |
| DIN | ESP32 GPIO41 |
| SD | 3V3 |
| GAIN | 悬空 |
| SPK+ | 喇叭任意一根线 |
| SPK- | 喇叭另一根线 |

喇叭两根线仅接 SPK+、SPK-，不能接 GND。

### 两个 MAX9814 模拟麦克风（需新增 ADS1115）

MAX9814 的 `OUT` 是模拟音频，不能接 ESP32 或 SC16IS752 的普通 GPIO。使用 ADS1115 把两个模拟输出转换为 I2C 数据；ADS1115 放在 TCA CH3。

| ADS1115 引脚 | 连接到 |
|---|---|
| VDD | 3V3 |
| GND | GND |
| SDA | TCA CH3 SD3 |
| SCL | TCA CH3 SC3 |
| ADDR | GND（I2C 地址 `0x48`） |
| REF（本板丝印；即 ADS1115 的 ALERT/RDY） | 暂不接；它是开漏告警/转换完成输出，不是电压基准 |
| AIN0 | MAX9814 #1 OUT |
| AIN1 | MAX9814 #2 OUT |
| AIN2、AIN3 | 暂不使用，接 GND |

| 每个 MAX9814 引脚 | 连接到 |
|---|---|
| VDD / VCC | 3V3 |
| GND | GND |
| OUT | #1 接 ADS1115 AIN0；#2 接 ADS1115 AIN1 |
| GAIN | 接 GND，50dB 总增益；若环境很安静可改为悬空，60dB |
| AR | 接 GND，使用较快的 1:500 攻击/释放比 |

ADS1115 的最高采样率为 860 SPS，适合音量、拍手/声音事件和低频方向判断；不适合录制语音。语音采集继续使用 INMP441。

## 6. Y01 与 TOF200F（通过 SC16IS752）

### SC16IS752 接线

下表按 I2C 模式接线；图中的 `I2C/SPI` 必须为高电平，才能选择 I2C。

| SC16IS752 功能引脚 | 连接到 |
|---|---|
| VCC | 3V3 |
| GND / VSS | GND |
| RESET | 3V3（保持非复位状态） |
| I2C/SPI | 3V3（选择 I2C） |
| A0/CS | GND（I2C 地址选择） |
| A1/SI | GND（I2C 地址选择） |
| SCL/SCLK | TCA CH0 SC0 |
| SDA/VSS | TCA CH0 SD0 |
| NC/SO | 不接 |
| IRQ | 暂不接，程序轮询 FIFO |
| TXA / RXA | 接 Y01 |
| TXB / RXB | 接 TOF200F |

| GP0 | 预留 |
| GP1 | LED1 输出 |
| GP2 | 继电器控制输出（经继电器驱动模块或三极管/MOSFET） |
| GP3 | 预留 |
| GP4 | LED2 输出 |
| GP5 | MAX30102 INT 输入 |
| GP6 | 不接 |
| GP7 | 黑帽按键 S 输入 |

## 7. 3.5 寸 ILI9488 SPI 屏与电容触摸

### TFT 显示部分

| 屏幕引脚 | 连接到 |
|---|---|
| VCC | 5V（按屏幕转接板标注） |
| GND | GND |
| SCK / CLK | ESP32 GPIO4 |
| MOSI / SDI | ESP32 GPIO5 |
| CS | ESP32 GPIO6 |
| DC / RS | ESP32 GPIO7 |
| RST / RESET | ESP32 GPIO10 |
| BL / LED | 外部三极管/MOSFET背光驱动输出；驱动控制端接 ESP32 GPIO11 |
| MISO / SDO | 不接（仅 ILI9488 显示控制器不回读像素数据） |
| IM0 | 3V3 / HIGH |
| IM1 | 3V3 / HIGH |
| IM2 | 3V3 / HIGH |

`IM0/IM1/IM2 = 111` 才会选择 ILI9488 的 4-line 8-bit SPI 接口。若转接板已经通过电阻或焊桥固定为 4SPI，不需要重复外接；若三个模式脚或焊桥可配置，必须确认三者均为高电平，否则控制器会把 SCK/MOSI/DC 当成其他接口而一直白屏。

GPIO11 只输出背光开关控制信号。由于 BL/LED 实测会将直接连接的 GPIO
压到约 1.7V，不得再由 GPIO11 直接给背光供电；应根据转接板 BL 极性增加
三极管或逻辑级 MOSFET 驱动。软件通过 `ili9488_display_set_backlight()`
控制 GPIO11，后续可用于无操作定时熄屏。

### 电容触摸部分

| 触摸引脚 | 连接到 |
|---|---|
| SDA | 直连 ESP32 GPIO8 |
| SCL | 直连 ESP32 GPIO9 |
| RST | ESP32 GPIO12 |
| NC | 不接 |
| NT | ESP32 GPIO13（也可在软件中仅轮询 SDA/SCL） |

屏幕供电为 5V 时，先测量触摸 SDA、SCL 的空闲电平：只有约 3.3V 才可直连 ESP32；若为 5V，必须增加双向 I2C 电平转换模块。

## 8. I2C 视觉识别模块

| 模块引脚 | 连接到 |
|---|---|
| VCC / VIN | 按模块标注供电 |
| GND | 公共 GND |
| SCL | 直连 ESP32 GPIO9 |
| SDA | 直连 ESP32 GPIO8 |

视觉模块的 SDA、SCL 空闲电平必须约为 3.3V；如果模块侧为 5V I2C，
必须增加双向 I2C 电平转换。若模块还有 INT、RST 等引脚，确认具体型号后
再从 GPIO14、GPIO15、GPIO18、GPIO21 中分配，不要先随意连接。

## 10. 供电总览

| 设备 | 建议供电 |
|---|---|
| ESP32-S3 | USB |
| DS3231、SHT30、INA219、MAX30102、OLED、TCA9548A、SC16IS752、PCA9685、ADS1115、MAX9814 | 3V3 |
| I2C 视觉识别模块 | 按模块标注供电；I2C 逻辑必须为 3.3V |
| 3.5 寸屏、MAX98357、Y01 | 稳定外部 5V |
| 总线舵机 | 独立舵机电源 |
| 小车电机 | 独立电机电源，接 TB6612FNG 的 VM |

不要将两个独立的 3.3V 稳压器输出直接并联；不同电源的负极统一接公共 GND。
