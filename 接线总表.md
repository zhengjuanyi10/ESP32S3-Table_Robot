# 桌面机器人模块接线总表

> 最终架构：ESP32-S3 → TCA9548A（I2C 扩展）→ 传感器 / OLED / 摄像头 / 触摸 / UART 扩展 / 小车控制。
>
> 需要额外准备：SC16IS752 双 UART 模块、PCA9685 PWM 模块、TB6612FNG 双电机驱动模块、ADS1115 四路 ADC 模块（用于两个 MAX9814）。
>
> 所有电源的 GND 必须共地。ESP32 GPIO 只能接 3.3V 逻辑。

## 1. ESP32-S3 原生 GPIO 总览

| 功能 | GPIO | 连接对象 |
|---|---:|---|
| I2C SDA | GPIO8 | TCA9548A SDA |
| I2C SCL | GPIO9 | TCA9548A SCL |
| 舵机总线 TX | GPIO17 | ZLink RX |
| 舵机总线 RX | GPIO16 | ZLink TX |
| I2S 麦克风数据输入 | GPIO1 | INMP441 SD |
| I2S 位时钟 | GPIO19 | INMP441 SCK、MAX98357 BCLK |
| I2S 左右声道时钟 | GPIO20 | INMP441 WS、MAX98357 LRC |
| I2S 音频数据输出 | GPIO41 | MAX98357 DIN |
| 触摸复位 | GPIO21 | 3.5 寸屏触摸 RST |
| TFT 背光 | GPIO38 | 3.5 寸屏 BL / LED |
| TFT SPI 时钟 | GPIO39 | 3.5 寸屏 SCK / CLK |
| TFT SPI MOSI | GPIO40 | 3.5 寸屏 MOSI / SDI |
| TFT SPI CS | GPIO42 | 3.5 寸屏 CS |
| TFT SPI DC | GPIO47 | 3.5 寸屏 DC / RS |
| TFT SPI 复位 | GPIO48 | 3.5 寸屏 RST / RESET |
| 串口日志 | GPIO43 / GPIO44 | 板载 USB-UART（COM3），不接外设 |

保留不接：GPIO0、GPIO3、GPIO45、GPIO46、GPIO26 至 GPIO37。

## 2. TCA9548A I2C 扩展板

### 与 ESP32-S3 接线

| TCA9548A 引脚 | 连接到 |

| VCC / VIN | ESP32 3V3 |
| GND | ESP32 GND |
| SDA | ESP32 GPIO8 |
| SCL | ESP32 GPIO9 |
| A0、A1、A2 | GND（地址 0x70） |
| RST / RESET（若有） | 3V3 |

### 通道分配

| TCA 通道 | 模块 |
|---|---|
| CH0（SD0 / SC0） | DS3231、SHT30 |
| CH1（SD1 / SC1） | 3.5 寸屏电容触摸 |
| CH2（SD2 / SC2） | INA219、MAX30102 |
| CH3（SD3 / SC3） | ADS1115（两个 MAX9814 模拟麦克风） |
| CH4（SD4 / SC4） | OLED |
| CH5（SD5 / SC5） | OV5640 摄像头 |
| CH6（SD6 / SC6） | SC16IS752 双 UART 扩展模块 |
| CH7（SD7 / SC7） | PCA9685 小车控制 PWM 模块 |

## 5. 音频模块

| INMP441 引脚 | 连接到 |
|---|---|
| VDD | 3V3 |
| GND | GND |
| SCK | ESP32 GPIO19 |
| WS | ESP32 GPIO20 |
| SD | ESP32 GPIO1 |
| L/R | GND |

### MAX98357 功放和喇叭

| MAX98357 引脚 | 连接到 |
|---|---|
| VIN / VCC | 外部稳定 5V |
| GND | GND |
| BCLK | ESP32 GPIO19 |
| LRC / LRCK / WS | ESP32 GPIO20 |
| DIN | ESP32 GPIO41 |
| SD | 3V3 |

| SPK+ | 喇叭任意一根线 |
| SPK- | 喇叭另一根线 |

喇叭两根线仅接 SPK+、SPK-，不能接 GND。

### 两个 MAX9814 模拟麦克风（需新增 ADS1115）

MAX9814 的 `OUT` 是模拟音频，不能接 ESP32 或 SC16IS752 的普通 GPIO。使用 ADS1115 把两个模拟输出转换为 I2C 数据；ADS1115 放在 TCA CH3。

| ADS1115 引脚 | 连接到 |
|---|---|
| VDD | 3V3 |
| GND | GND |
| SDA | TCA CH3 SD3 |
| SCL | TCA CH3 SC3 |
| ADDR | GND（I2C 地址 `0x48`） |
| REF（本板丝印；即 ADS1115 的 ALERT/RDY） | 暂不接；它是开漏告警/转换完成输出，不是电压基准 |
| AIN0 | MAX9814 #1 OUT |
| AIN1 | MAX9814 #2 OUT |
| AIN2、AIN3 | 暂不使用，接 GND |

| 每个 MAX9814 引脚 | 连接到 |
|---|---|
| VDD / VCC | 3V3 |
| GND | GND |
| OUT | #1 接 ADS1115 AIN0；#2 接 ADS1115 AIN1 |
| GAIN | 接 GND，50dB 总增益；若环境很安静可改为悬空，60dB |
| AR | 接 GND，使用较快的 1:500 攻击/释放比 |

ADS1115 的最高采样率为 860 SPS，适合音量、拍手/声音事件和低频方向判断；不适合录制语音。语音采集继续使用 INMP441。

## 6. Y01 与 TOF200F（通过 SC16IS752）

### SC16IS752 接线

下表按 I2C 模式接线；图中的 `I2C/SPI` 必须为高电平，才能选择 I2C。

| SC16IS752 功能引脚 | 连接到 |
|---|---|
| VCC | 3V3 |
| GND / VSS | GND |
| RESET | 3V3（保持非复位状态） |
| I2C/SPI | 3V3（选择 I2C） |
| A0/CS | GND（I2C 地址选择） |
| A1/SI | GND（I2C 地址选择） |
| SCL/SCLK | TCA CH6 SC6 |
| SDA/VSS | TCA CH6 SD6 |
| NC/SO | 不接 |
| IRQ | 暂不接，程序轮询 FIFO |
| TXA / RXA | 接 Y01 |
| TXB / RXB | 接 TOF200F |
| RTSA、CTSA、RTSB、CTSB | 不接，不使用硬件流控 |
| GP0 | 3.5 寸电容触摸 NT（触摸中断输入） |
| GP1 | LED1 输出 |
| GP2 | 继电器控制输出（经继电器驱动模块或三极管/MOSFET） |
| GP3 | 预留 |
| GP4 | LED2 输出 |
| GP5 | MAX30102 INT 输入 |
| GP6 | 不接 |
| GP7 | 黑帽按键 S 输入 |

> 如果这是裸芯片而不是带晶振、去耦电容和排针的成品模块，请购买成品 SC16IS752 I2C 双 UART 模块；裸芯片还需要外围时钟与电源去耦电路，不能直接用杜邦线连接。

### Y01 空气质量模块

| Y01 引脚 | 连接到 |
|---|---|
| VCC | 5V |
| GND | GND |
| A（模块 RX） | SC16IS752 TXA |
| B（模块 TX） | SC16IS752 RXA |

### TOF200F 测距模块

| TOF200F 引脚 | 连接到 |
|---|---|
| VCC | 按模块丝印供电 |
| GND | GND |
| RX | SC16IS752 TXB |
| TX | SC16IS752 RXB |
| SDA | 不接（本方案使用 UART） |
| SCL | 不接（本方案使用 UART） |

若 Y01 或 TOF200F 的 UART 电平高于 3.3V，必须在 UART 线增加电平转换模块。

### 电容触摸部分

| 触摸引脚 | 连接到 |
|---|---|
| SDA | TCA CH1 SD1 |
| SCL | TCA CH1 SC1 |
| RST | ESP32 GPIO21 |

## 10. 供电总览

| 设备 | 建议供电 |
|---|---|
| ESP32-S3 | USB |
| DS3231、SHT30、INA219、MAX30102、OLED、TCA9548A、SC16IS752、PCA9685、ADS1115、MAX9814 | 3V3 |
| OV5640 | 稳定 3.3V，建议预留 ≥500mA |
| 3.5 寸屏、MAX98357、Y01 | 稳定外部 5V |
| 总线舵机 | 独立舵机电源 |
| 小车电机 | 独立电机电源，接 TB6612FNG 的 VM |

不要将两个独立的 3.3V 稳压器输出直接并联；不同电源的负极统一接公共 GND。
